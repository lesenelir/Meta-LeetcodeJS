# 动态规划

具有多个重叠子问题时，使用动态规划来处理。
动态规划具有状态，当前的状态是由上一个状态推导出来的，而贪心没有推导状态，直接从局部直接选取最优解

动态规划和贪心区别：
- 动态规划是由上一个状态推导出来的
- 贪心算法是直接在局部选取最优解


定义：将大规模问题转换为小规模问题，并缓存中间结果


递归：  自己调用自己（用来枚举所有情况） =  优化 ： 动态规划，进行记忆

- 背包问题

- 股票问题
- 子序列问题
